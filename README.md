## Setup Project
- download repository ini
- composer install
- npm install
- Bikin file .env (copy isinya dari .env.example)
- Siapkan database (mysql) sesuaikan namanya DB, user, pass di file .env sebelumnya
- jalankan "php artisan serve", hasil di localhost:8000
